# Objective Journal - 客观日记系统

## 核心理念

**只记录事实，不带情绪。** 通过长期客观记录，观察自己的生活模式。

参考马伯庸日记法：时间、地点、人物、事件——不需要"今天心情很好"这类主观评价。

## 数据存储

所有记录存储在 Obsidian Vault：
- **路径**：`D:\ObsidianVault\日记\`
- **结构**：
  ```
  日记/
  ├── 2026/
  │   ├── 2026-05-22.md（日常记录）
  │   └── ...
  ├── 金句/
  │   └── 2026-05-22-原则.md
  ├── 启发/
  │   └── 2026-05-22-决策方法.md
  └── 睡眠/
      └── 2026-05-22.md
  ```

## 功能模块

### 1. 日常客观记录

**输入方式**：Flomo + 标签 `#日志/事实`

**自动解析字段**：
| 字段 | 示例 |
|------|------|
| 时间 | 15:00 |
| 地点 | 星巴克 |
| 人物 | 国栋 |
| 事件 | 喝茶聊天 |
| 话题 | 孩子教育 |

**解析脚本**：`scripts/parse_memo.py`

**示例**：
```
输入：下午3点和国栋在星巴克喝茶聊孩子教育 #日志/事实
解析：
- 时间：15:00
- 地点：星巴克
- 人物：国栋
- 事件：喝茶聊天
- 话题：孩子教育
```

### 2. Linter 情绪词过滤

**触发条件**：检测到 `#日志/事实` 标签时自动运行

**检测逻辑**：`scripts/linter.py`

**反向词表**：
```python
EMOTION_WORDS = {
    # 负面
    "糟糕", "愤怒", "讨厌", "极其", "太慢了", "崩溃", "烦死", 
    "恶心", "无语", "绝望", "压抑",
    # 正面
    "开心", "太棒了", "超级", "完美", "幸福死了", "爽",
    # 程度副词
    "极其", "非常", "特别", "超级", "太...了"
}
```

**处理方式**：
1. 即时提醒：在记录下方追加提示
2. 周复盘标红：统计"情绪词污染率"

**输出示例**：
```
⚠️ 检测到情绪词：[极其、糟糕]
💡 提示：试试改成客观描述，如"会议用时2小时，未达成决议"
```

### 3. 金句/启发记录

**标签**：`#输入/金句`、`#输入/启发`

**自动提取**：
- 来源（书名/播客/文章）
- 原文引用
- 个人理解（可选）

**示例**：
```
输入：读《原则》看到一句话：痛苦+反思=进步 #输入/金句 #原则
存储：
---
来源：书籍《原则》
金句：痛苦+反思=进步
标签：#原则
日期：2026-05-22
---
```

### 4. 睡眠质量记录

**标签**：`#睡眠`

**输入格式**：
- 自然语言："昨晚11点半睡的，7点起，质量一般 #睡眠"
- 结构化："睡眠 23:30-07:15 3分 半夜醒了两次 #睡眠"

**自动计算**：睡眠时长

### 5. 基础搜索

**搜索脚本**：`scripts/search.py`

**支持维度**：
- 按时间："上周三做了什么"
- 按人物："最近见了哪些人"
- 按地点："去过哪些咖啡店"
- 按事件类型："最近开过哪些会"
- 组合搜索："上个月和王总相关的会议"

### 6. 每周自动复盘

**触发**：Cron 定时任务（每周日 21:00）

**复盘内容**：
1. **时间分布**：本周时间花在哪里
2. **社交网络**：见了哪些人，聊了什么
3. **知识网格**：本周关注的知识领域（基于金句/启发）
4. **知行转化**：启发 → Action Item 的转化率
5. **情绪词统计**：污染词出现次数
6. **下周建议**：基于数据的行动建议

**复盘脚本**：`scripts/weekly_review.py`

### 7. 时间轴视图

**可视化页面**：`assets/timeline.html`

**功能**：
- 按时间顺序展示所有记录
- 筛选：按人物/地点/事件类型/时间段
- 统计概览：记录条数、见过的人、去过的地方、情绪词污染率
- 情绪词标记：红色高亮显示

**打开方式**：浏览器打开 `D:\Skill\objective-journal\assets\timeline.html`

### 8. 事实验证

**触发条件**：检测到模糊记录时主动提示

**模糊模式**：
- 时间模糊："下午"、"晚上"、"最近"
- 人物模糊："开会"、"见面"（未提及具体人名）
- 事件模糊："讨论了一下"、"处理了一些事"

**提示模板**：
```
📝 记录已保存！建议补充：
❓ 时间：具体是几点？
❓ 人物：和谁？
❓ 结果：有什么结论？

回复即可补充。
```

## 使用流程

### 日常记录
1. 用户发 Flomo：`下午3点和国栋喝茶聊孩子教育 #日志/事实`
2. QClaw 检测标签 → 运行 Linter
3. 解析内容 → 存入 Obsidian
4. 如有情绪词 → 追加提示

### 周复盘
1. Cron 触发（每周日 21:00）
2. 读取本周所有记录
3. 生成复盘报告 → 存入 Obsidian
4. 推送给用户
