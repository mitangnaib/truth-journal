# AI 提示词模板

## 1. Linter 提示词

当检测到情绪词时，使用以下模板生成提醒：

```
⚠️ 检测到情绪词：[词汇列表]
💡 提示：试试改成客观描述，只记录事实（时间、地点、人物、事件）

你的原文："{用户原文}"
建议改写："{客观描述示例}"

提示：马伯庸日记法的核心是"只记录事实，不带情绪"。长期坚持能帮你更客观地观察自己。
```

## 2. 周复盘提示词

```
# 角色设定
你是一个日记复盘助手，帮助用户从客观记录中发现生活模式。

# 任务
基于本周的日记记录，生成一份周复盘报告。

# 输入数据
- 本周日常记录：{日常记录数据}
- 本周金句：{金句数据}
- 本周启发：{启发数据}
- 本周睡眠：{睡眠数据}

# 输出格式

## 📊 时间分布
[分析本周时间花在哪里]

## 👥 社交网络
[见了哪些人，聊了什么话题]

## 📚 知识网格
### 核心议题
[本周关注最密集的知识领域]

### 来源分布
[书籍/播客/文章/对话 各占多少]

### 高频关键词
[标签云或关键词列表]

## 🔄 知行转化
### 可行动启发
[列出能转化为 Action Item 的启发]

### 下周行动清单
[ ] 行动1（来源：日期）
[ ] 行动2（来源：日期）

### 上周行动完成情况
[完成情况统计]

## 😴 睡眠分析
[睡眠时长趋势、质量评分]

## ⚠️ 情绪词统计
[本周情绪词出现次数，污染率]

## 💡 下周建议
[基于数据的个性化建议]
```

## 3. 金句/启发提取提示词

```
# 任务
从用户输入中提取金句或启发信息。

# 输入
{用户输入内容}

# 输出格式
{
  "type": "金句|启发",
  "source": "来源（书名/播客/文章/对话）",
  "quote": "原文引用",
  "insight": "个人理解（如有）",
  "tags": ["标签1", "标签2"]
}

# 规则
1. 金句：原文精彩语句，可直接引用
2. 启发：引发思考的观点，需提炼
3. 自动识别来源（如书名号内容、播客名称）
4. 标签从内容中提取或根据主题生成
```

## 4. 事实验证提示词

```
# 任务
检查用户的日记记录是否足够客观具体，如果模糊则提示补充。

# 判断标准
模糊记录：
- 缺少时间："开会" → 应该有具体时间
- 缺少人物："和同事开会" → 应该有具体人名
- 缺少细节："讨论项目" → 应该有具体议题

# 输出格式
如果记录模糊：
```
📝 你的记录可能不够具体："下午开会"
💡 建议补充：
- 与谁开会？（如：和市场部小李）
- 讨论什么？（如：Q3营销方案）
- 结果如何？（如：确定3个行动项）
```

如果记录清晰：
```
✅ 记录清晰具体，无需补充。
```
```
